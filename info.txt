--- Running ios emulator ---
ionic capacitor add ios - Generate the native project
ionic capacitor open ios - Open the project in Xcode.
ionic capacitor copy ios - copy changes to native project
ionic capacitor update ios - install capacitor plugins 

ionic capacitor build ios - do copy and then open
ionic capacitor sync ios - do copy and update

npx cap add ios - Generate the native project
npx cap copy ios - copy changes to native project
npx cap update ios - update ios plugins
npx cap sync ios - command runs copy and then update
npx cap open ios -  Open the project in Xcode.

npx cap build - build APK
